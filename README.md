# 💼 Job Board Application

A full-stack Job Portal built using **AngularJS, PHP, and MySQL**.

This application allows job seekers to search and apply for jobs, recruiters to post jobs, and admins to manage the system.

---

## 🚀 Features

### 👨‍💼 Job Seeker
- Search jobs with filters (skill & location)
- Apply for jobs
- ❤️ Save / Unsave jobs (LinkedIn-style)
- Upload resume
- View applied jobs

### 🏢 Recruiter
- Post new jobs
- Manage job listings
- View applications
- Approve / Reject candidates

### 🛠 Admin
- Manage users
- Manage recruiters
- Manage jobs
- View reports

---

## 🧰 Tech Stack

- **Frontend:** AngularJS
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (XAMPP)
